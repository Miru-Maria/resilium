## 30-Minute Monthly Ritual

Run on the **first Monday of each month**. Log all findings in the Change Log below.

| Time | Task | What to watch for |
|---|---|---|
| 0–5 min | Check Google Alert email digest | Press coverage, blog posts, product announcements since last month |
| 5–10 min | Open BetterUp Ready™ page + LinkedIn jobs | New job titles: "consumer," "B2C," "self-service" = pivot risk. New features on Ready™ page? |
| 10–14 min | Open ReadyScore homepage (curl if needed) | Changed framing? New categories added beyond 11? Signs of Pro tier? Mobile app? |
| 14–17 min | Open Coached.com homepage | Any return to resilience positioning? New pricing tier? |
| 17–20 min | Open Driven Resilience shop | New USD pricing? US-targeted content? Consumer app launch? |
| 20–24 min | Search Product Hunt: `resilience score` `preparedness app` `emergency planning` | Any new consumer entrant not caught by Google Alerts |
| 24–27 min | Run curl on any new entrant homepage found | Apply SPA research chain if JS-rendered: meta tags → JS bundle extraction |
| 27–30 min | Update Change Log below | One row per finding. Flag escalation triggers with ⚠️ |
